Sobre el proyecto

Desarrollo web con enfoque profesional

Este proyecto fue desarrollado como parte de la asignatura Programación III, integrando tecnologías modernas para construir un sitio web corporativo funcional, intuitivo y orientado a ofrecer una excelente experiencia al usuario.

Durante su desarrollo se aplicaron principios de diseño responsivo, organización del código, integración de servicios externos y buenas prácticas de programación para obtener un resultado profesional.
